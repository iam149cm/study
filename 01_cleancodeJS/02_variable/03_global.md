## 전역공간 사용 최소화
Q. 전역공간을 왜 사용하면 안되는가? (왜 최소화 하여 사용해야 하는가?)

- 전역공간이란 최상위를 의미하며 Global / Window 로 나뉜다. 

|Global|Window|
|:---:|:---:|
|Node JS 환경|브라우저 환경|

## 위험요소
- 파일을 나누면 코드의 구역 (스코프)도 나뉘는 게 아닌가?
- 그렇지 않다! 
- 전역공간 오염을 예방하는 방법 : IIFE, Module, Closure, const/let 사용

## 전역공간을 더럽히면 안된다
- 이유 : 어디서나 접근이 가능하다.
- 런타임 환경에서 분리가 되어 있지 않음. 

Q. 어떻게 예방 ? 
1. 전역변수를 만들지 않는다
2. 지역변수만 만든다
3. window / global 을 조작하지 않는다.
4. const / let 사용
5. IIFE, Module, closure 를 통해 스코프를 나눈다.
