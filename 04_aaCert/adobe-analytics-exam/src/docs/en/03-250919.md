### ‚úÖ Mini Concept Learning: Consent Management & Data Mapping

#### ‚òùüèª What is Consent Management?  
A framework for managing user consent regarding personal data and cookies.  
- Opt-in: All data can be collected  
- Partial Consent: Analytics allowed, Advertising blocked  
- Opt-out: Cookies and personalized data collection blocked  
- ref link : https://experienceleague.adobe.com/en/docs/id-service/using/implementation/opt-in-service/optin-overview  

#### ‚úåüèª Privacy Service (GDPR/CCPA)  
- Delete Request: Deletes all Adobe Analytics data of a specific visitor  
- Access Request: Provides the visitor with a download of their own data  
- Opt-out Handling: Deletes ECID and stops further data collection  
- ref link : https://experienceleague.adobe.com/en/docs/experience-platform/privacy/home  

#### üëâüèª Data Mapping  
Mapping consent levels to variables (eVar/prop/event) using Processing Rules / Data Elements.  
- Example: `Consent_Status = Opt-in / Opt-out` ‚Üí Enables reporting segmented by consent level  
- Practical note: If data does not appear in reports, always check for consent refusal first.  
- ref link :  
    - Processing Rules overview : https://experienceleague.adobe.com/en/docs/analytics/admin/admin-tools/manage-report-suites/edit-report-suite/report-suite-general/processing-rules/pr-overview

### üìù Practical Points
- If consent is refused, Adobe Analytics hits are not sent at all.  
- Partial consent is usually handled in Launch rules: advertising tags are blocked, basic analytics allowed.  
- GDPR/CCPA requests are processed through Adobe Privacy Service.

### üéØ Practice Questions

**Q1.** In Adobe Analytics, what happens if a user selects *Opt-out* in the consent banner?  

    A. Only advertising data is blocked  
    B. No Analytics hit is collected  
    C. Data is anonymized automatically  
    D. ECID is duplicated  

**Answer: B**  
**Explanation:** Opt-out is a full refusal, so no Analytics hit is sent to the server.  

---

**Q2.** Which Adobe service is responsible for handling GDPR/CCPA delete requests?  

    A. Launch  
    B. Privacy Service  
    C. Audience Manager  
    D. Target  

**Answer: B**  
**Explanation:** Adobe Privacy Service is in charge of handling GDPR/CCPA-related data deletion and access requests.  

---

**Q3.** Why is Data Mapping of Consent status important in reporting?  

    A. It ensures ECID is regenerated  
    B. It allows segmentation/filtering by consent level  
    C. It improves pathing analysis  
    D. It triggers Adobe Target activities  

**Answer: B**  
**Explanation:** Mapping consent status to variables allows reports to distinguish and analyze users based on their consent level.
