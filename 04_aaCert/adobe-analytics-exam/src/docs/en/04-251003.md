**Q1.** What is the key conceptual difference between an eVar and a prop?  

    A. eVars expire at page view  
    B. props enable attribution  
    C. props allow pathing analysis  
    D. eVars allow pathing analysis

**Q2.** Which variable type should be used to measure a purchase completion action?  

    A. eVar  
    B. prop  
    C. event  
    D. segment  

**Answer: C**  
**Explanation:** Actions like purchase completions should be tracked as Success Events to measure KPIs.

**Q3.** When defining KPIs in an SDR, why is it important to clearly distinguish eVar, prop, and event?

**Answer: B**  
**Explanation:** Clear separation of roles prevents reporting errors. Proper KPI design requires distinct definitions of eVar, prop, and event in the SDR.
